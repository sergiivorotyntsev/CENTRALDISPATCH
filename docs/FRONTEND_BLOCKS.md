## How to Report Issues

When reporting an issue, please include:

1. **Block ID** - e.g., "DOCS-UPLOAD" or "REV-ITEM"
2. **Expected behavior** - What should happen
3. **Actual behavior** - What actually happens
4. **Steps to reproduce** - How to trigger the issue
5. **Screenshot** (if applicable)

### Example Issue Report:

```
Block: DOCS-LIST
Expected: Clicking delete should remove document from list
Actual: Document disappears but reappears after page refresh
Steps:
1. Go to Documents page
2. Click delete on any document
3. Observe it disappears
4. Refresh page
5. Document is back
```
